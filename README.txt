How to run:
(1)press start to start collapse a level, edge and face different will be render at real time
(2)press load to load all level obj into mesh
(3)drag the slide to see the different level
(4)check the checkbox to use camera to decide which level to render. 

My program will continue to run untill there are no eligible edge to collapse.
